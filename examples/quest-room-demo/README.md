# Quest Room Demo

Future example project for demonstrating the full observe-act-verify loop on a Meta XR scene.

Target scenario:

1. Open a small Quest-ready room scene.
2. Capture Scene View and Game View screenshots.
3. Validate Meta XR setup.
4. Detect console/configuration issues.
5. Apply controlled fixes.
6. Verify logs, screenshots, and build readiness.

This folder intentionally starts as documentation only. The Unity example project should be added once the plugin bridge is stable.
